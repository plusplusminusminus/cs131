5) What is the init process in Linux?
ANSWER:
The init process is the first process that is run when a Linux system boots. It is the parent of all other processes and is responsible for starting and stopping them.

6) Explain the command in not more than 4 sentences: ps aux

ANSWER:
The `ps aux` command displays information about all processes currently running on the Linux system. It shows a detailed list of processes, incuding the user who owns the process, the Process ID, CPU and memory usage, and the command used to start the process. The 'a' option causes ps to display information for all processes; the 'u' option tells ps to displayed detaied information about each process that is running.
